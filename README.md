# hash-cryptography
Programa em Python para criptografia usando hash (SHA256, MD5, SHA512, SHA1)


Este projeto implementa criptografia em hash usando Python. 
Recursos
- Suporte a múltiplos algoritmos (SHA256, MD5, SHA512)
- Entrada de texto e saída em hash
- Fácil integração em outros projetos.
